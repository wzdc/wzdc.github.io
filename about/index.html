# 关于我

性格内向，计算机专业学生，从小学到初中学习都很差的学生（因为没兴趣所以不听课，现在后悔莫及），喜欢计算机专业、二次元、打游戏 (很少)。初中毕业后就去了技校学习计算机，到了技校就没有像初中那样不听课了（也有一些课没听），考试名次能进前五，感觉我现在读的学校就很垃圾 。

# 名字由来

“无知的错” 这个名字的由来是我有一个 QQ 好友他叫 “无知的错、”，有一天他改名了我就用了他以前的名字和头像，当时他还问我 “你为什么用我以前的名字和头像？”，这个名字一用就用了好几年，头像只要想换的时候就会从网上找，我想自己取一个重名率低、好听的网名，但是我不会取呀。这个名字重名还是比较少的。

# 搭建网站

初次建站是通过网上的自助建站平台建站的，当时我还是个 xxs。那时候开发了一个网页很激动，日日夜夜都在想我的网站，还记得我那时候五六点起来设计网页。后来会编程后就没再用了。

# 编程

在初中的时候我想开一个 MC 服务器，朋友给我推荐了某某云。偶然一天发现某某云可以创建免费站点，于是我就点击创建了一个。初次见到代码，我看不懂就改了一些中文字，又去了网上搜了一些网页开发的教程自学。

# 开发安卓 APP

当时我用过一个名叫 iapp 的编程软件，上手容易，代码也容易记（因为代码短）。现在没在用了，打算自学 Java 来开发 APP。原因是因为：

  1. 导入别人的源码容易中病毒
  2. 写个 APP 太烧钱（很多功能要 VIP）

# 最后

因为兴趣爱好选择了计算机，我会努力学习计算机。一路上很容易遇到知识盲区，但是会去慢慢理解（实在不会就去搜索一下或群里问问一些大佬）。因为什么基础都没打好，只能以后慢慢弥补..
