首页 API

又一个随机图片 API，之前也写过一个随机图 API，是调用搜索引擎的搜图 API 返回的图片。每次打开 B 站首页都有好看的动漫图，这次直接调用 B 站的文章搜索接口做随机图，这是我前几天写的，因为偶尔会返回一些装饰图片，所以感觉不完美就扔 moe team telegram group File 里了，昨天无意间想到了可以通过图片分辨率来判断是不是装饰图，特意完善了一下！（代码写的烂，大佬勿喷）

# 为什么做这个？

  1. 因为 B 站上的动漫图比搜索引擎搜到的动漫图好看
  2. 搜索引擎搜的是全网的图片，所以难免会遇到原站的图片无法打开或打开速度慢的问题
