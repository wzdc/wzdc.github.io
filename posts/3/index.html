我做机器人的经历

# 起初

（我没记错的话）我看到我的好友群里有一个 QQ 机器人，我就问好友是怎么弄的，群主就给我推荐了一个资源网站，我在那个网站上下载了一个手机版的机器人，手机版的也挺好，这样就不需要购买主机挂机器人了。这个软件功能太少了，和好友群里的机器人对比一点都不一样。之后我换了好多个机器人软件，用过酷 Q（因为不会用就卸了）等软件（具体不太记得了）。

# 刷赞机器人

我看到好友群里有一个机器人会刷名片赞，新入群和每天签到会送名片赞，我也想搞就问好友怎么弄，好友也和我说了用什么机器人，还要买授权、名片赞，之后我买完后朋友就帮我弄了，我就去其他群发我的群送名片赞，结果被踢了，但是还是搞来了一些人。

# 题外话：名片赞

当时看到别人网站有卖名片赞的，我也想弄，我的好友就让我对接什么平台，那个平台名片赞价格低，直接中间商赚差价。我小号多想自营，但是我不会弄，我问好友好友也没有明确的答复。

# 晨风 QQ 机器人

用了那么多 QQ 机器人软件，最终我选择了晨风 QQ 机器人，而且最后的几年里也一直在用晨风 QQ 机器人

# 手机版晨风 QQ 机器人（疑似盗号软件）

我之前在网上找到了一个手机版晨风 QQ 机器人，因为用手机挂就不用租台电脑挂了，登录试了好多次都显示登录失败，质疑他是盗号软件是因为我看到一个盗号软件生成器的软件他可以生成手机版晨风 QQ 机器人的盗号软件

# 起初

我用的是破解版，词库是网上搜的，起初我想拿这个机器人管群，机器人入群后，开启聊天功能群确实活跃了不少，有的群友喜欢机器人，有的群友不喜欢，也有很多人退群（估计是太吵了），于是就造成了有时开娱乐功能，有时不开。我又发投票，最终我决定开娱乐功能，群默认关闭聊天功能，私聊机器人是可以单独和机器人聊天的。

我在使用破解版的时候，遇到过 QQ 机器人发了一些奇怪的消息，当时我 QQ 机器人在群里发了 nmsl 、 .. 这种不正常的消息，群友就发了几个问号，我去查登录记录发现并没有异常登录记录，起初以为是远程账号被盗于是就改了远程账号，现在想起来觉得是破解版有后台。

# 之后

QQ 更新，管理员可以撤回成员消息了，晨风 QQ 机器人也随之更新支持撤回违禁词消息，因为我用的是破解版所以没有这种功能，于是我又去找能撤回消息的晨风 QQ 机器人破解版，但是没找到，在我犹豫了很多次后最终还是花了 200 元买了正版授权。

# 充值

# 起初

我开启娱乐功能是根据群友投票来的，他们很喜欢我的机器人，还专门搞了一个团队叫 临清 ，还莫名其妙收了一个徒弟（他一直叫我师傅，但我却一直不认可这个徒弟，因为我不想收徒）

# 之后

我的朋友也在玩机器人，他让我把他的金币调高，但是我不想破坏游戏平衡，于是就说你给我一块钱我给你一千万金币，他还真给我一块钱了，我也真给他金币了，于是就起内讧了。我们就开始搞充值赚钱了，赚到的钱五五分，价格如下

价格（人民币）数量游戏虚拟币
11000000金钱
110000威望

我们最贵的武器 / 道具只卖一千万金币（10 人民币），我们还把机器人租给其他人，5 元一个月（很划算吧！）

我还花了 50 元买了 codepay 的微信支付授权 (现在已经跑路了)、70 元买了彩虹代刷的授权 (现在已经停运了)，这样就不需要找我们人工充值了。使用了 PHP 写了一个推送系统，机器人每隔一分钟去服务器接收充值指令，如果有人充值不会立马到账，而是要等机器人去服务器上接收指令才会到账。服务器、授权费等这些费用都是我一个人出的。基本上就是帮别人充值和解决一些机器人问题。赚到多少呢？多的一天几十，少的就没人充值（感觉合伙人赚了我没赚呀）。

一路上遇到很多困难，（机器人账号被封、收钱账户被封）但我们都克服过来了

# 最后

还是不做了，不是腾讯封杀机器人我们才不做的，而是我不想做了，服务器是按量计费的，我想留点钱给我以后用。腾讯禁止第三方机器人，机器人账号被封了，机器人软件作者也跑路了，所以机器人不能用了（就算是已授权账号也用不了了）。我发现晨风 QQ 机器人论坛有破解版的机器人，于是就下载来继续用了。最后是在 2020 年快结束的时候我打算不做了，就问了合伙人还做吗，他也说不做了，我们就这样退出了。机器人号也被封了，要担保才能解封，不过后来还是担保解封了。解封后想再用机器人软件却发现框架不能用了（显示版本过低）。

因为我妈妈挺喜欢机器人的，我就想重操旧业，但是这次打算搞公益，数据还在，但是软件已经不能用了，如果还能用，我可能会重操旧业，做完全免费的公益机器人。

# 问 / 答

# 为什么聊起这个回忆？

是因为我在 GitHub 某仓库看到一个 issues，这个账号使用动漫头像，因为我喜欢二次元，看他头像可爱我就点开他的主页一看，大佬！在主页上我看到了他的邮箱，我就给他发了一封申请友链的邮件，于是等啊等啊等，本以为他不会回我了，有一天晚上我收到他给我回信，我很开心（当时我的友链页面没有搞好就没回了），我就加急搞友链页面了，搞好友链页面后就立马去回复邮件了，后来我们就加了 QQ 好友，于是聊呀聊就聊到了机器人，我就想把我当年做机器人的经历写下来。

# 为什么群有那么多人？

因为我在某些视频平台有一点点粉丝，而这个群就是一个粉丝群，是为了更好的管理群而添加的 QQ 机器人。二是因为之前送名片赞，所以有几个人加群。
