首页 API

调用百度、360、搜狗、神马图片搜索接口搜索一张图并返回。支持手机和电脑不同关键字搜索。 下载源码

注意：请求太快可能会被搜索引擎封禁或弹验证码哦！

# 请求

请求地址：https://api.wzdc.cc/image.php
请求方式：GET/POST
请求示例：https://api.wzdc.cc/image.php?source=baidu&type=download&search = 电脑二次元壁纸 & qsearch = 手机二次元壁纸

参数必填说明默认
search是要搜索的图片（例如电脑壁纸）
pesearch否手机访问时要搜索的图片（例如手机壁纸）（使用 q 参数传入的值）
source是搜索引擎（可选值：baidu、360、sougou、shenma）
type否数据返回类型（可选值参见 type 可选值)json
random否随机数范围1,100

# type 可选值

可选值说明
json压缩的 JSON 格式
img由服务器下载原站图片后再返回给客户端
thumburl跳转到搜索引擎保存的图片地址
redirect跳转到图片原站
download重定向到百度的图片下载 API

# 返回

参数数据类型说明
urlString原站图片链接
thumburlString搜索引擎保存的图片链接
downloadString百度图片下载接口链接
pint

  2. 非必要参数错误则传入的参数不会生效
  3. 必要参数有误则无法获取链接
